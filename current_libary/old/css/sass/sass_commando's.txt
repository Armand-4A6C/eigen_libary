=== Command Promt Commands ===

gem install sass <-- installs latest sass version

gem update sass <-- updates to latest sass version

sass -v <-- returns the installed version number

sass --watch "watchingdir":"outputdir" <-- automaticly compiles changes from the sass folder to the specified folder
