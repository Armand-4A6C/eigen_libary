##layout

Vbox vertical aligns items vertical
Borderpane has 5 regions top, bottom, left, center, and right
StackPane allows items to stack on eachother each item is added on top of the previous item
Gridpane  allows alignment with rows and collumns
FlowPane
AnchorPane
TilePane

Note: layout can be inserted into other layouts
